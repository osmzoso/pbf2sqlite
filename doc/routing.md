# Routing

To test the routing, a script **route.py** is available in the directory /test.

```
Usage:
./route.py DATABASE LON_START LAT_START LON_DEST LAT_DEST PERMIT CSVFILE

PERMIT: 1 (foot), 2 (bike), 4 (car) or 10 (bike_paved)
```
